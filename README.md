# Expense Tracker

A simple command-line Expense Tracker built using Python and MongoDB.

## Features

- Add Expense
- View Expenses
- Delete Expense by Date
- Automatically stores the current date
- MongoDB Database

## Technologies Used

- Python
- MongoDB
- PyMongo

## Project Structure

expense_tracker/
│── main.py
│── expense_tracker.py
│── db.py
│── README.md

## How to Run

1. Install PyMongo

pip install pymongo

2. Start MongoDB

3. Run

python main.py
